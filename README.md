# GERADOR DE SENHAS EM PYTHON

Projeto simples em python 
que gera uma senha de acordo com o que o usuário escolhe

## O QUE FAZ?
- Pergunta o tamanho de caracteres que a senha terá (mínimo de 8)
- Faz perguntas se terá letras, símbolos ou números
- Mostra dicas se caso o usuário recusa algum tipo
- Gera uma senha com base nas escolhas 

## Tecnologias utilizadas
- Python 3
- Bibliotecas padrões do python
    - 'random'
    - 'string'

## Como usar
1. Certifique-se de ter o python instalado
2. No terminal, abra a pasta do projeto
3. Execute o comando:

```bash
python main.py 
```

## Exemplo de uso
Quantos caracteres sua senha terá:8
Muito bem, vamos continuar.
Sua senha terá letras? responda apenas com sim ou não:sim
Sua senha terá números? novamente, apenas responda com sim ou não:não
Uma senha com números é mais segura, mas vamos continuar.
Sua senha terá símbolos, pela última vez, responda com sim ou não:sim
